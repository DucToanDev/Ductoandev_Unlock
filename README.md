# DucToanDev Premium Unlock

Module Surge/Shadowrocket mở khóa tính năng Premium cho các ứng dụng iOS.

## Tính năng

- **24 ứng dụng hỗ trợ**: Locket, YouTube, Spotify, SoundCloud, PicsArt, Emby, KineMaster, Lightroom, Photoshop, Djay, Calm, Bussu, VSCO, Mojo, Widgetsmith, Headspace, Funimate, Reface, Unfold, HTTPBot, 1Blocker, Structured, Splice, Facetune
- **Chặn quảng cáo**: YouTube, Spotify
- **Premium vĩnh viễn**: Các gói đăng ký không hết hạn

## Cài đặt

### Surge

1. Mở ứng dụng Surge
2. Vào **Module** > **Install Module**
3. Dán link module:
   ```
   https://raw.githubusercontent.com/DucToanDev/Ductoandev_Unlock/main/Module/DucToanDev.sgmodule
   ```

### Shadowrocket

1. Mở ứng dụng Shadowrocket
2. Vào **Config** > **Module**
3. Thêm link module tương tự

### Locket Widget

1. Tải file `locket.mobileconfig`
2. Cài đặt profile trên thiết bị iOS
